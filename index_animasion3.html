Пульсирующая Анимация

Демонстрация CSS-анимации pulse с различными цветами и эффектами. Каждый элемент пульсирует с уникальным ритмом и цветом.

.pulse {
  width: 80px;
  height: 80px;
  background: #ff6b6b;
  border-radius: 50%;
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0% { 
    transform: scale(0.9); 
    box-shadow: 0 0 0 0 rgba(255, 107, 107, 0.7); 
  }
  70% { 
    transform: scale(1); 
    box-shadow: 0 0 0 20px rgba(255, 107, 107, 0); 
  }
  100% { 
    transform: scale(0.9); 
    box-shadow: 0 0 0 0 rgba(255, 107, 107, 0); 
  }
}

Эта анимация создает эффект пульсации с помощью изменения масштаба и тени элемента. Она может использоваться для привлечения внимания к важным элементам интерфейса.
